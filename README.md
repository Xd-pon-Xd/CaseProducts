# 🛒 PantryBot — Telegram-бот для формирования списка покупок

PantryBot — это Telegram-бот, который автоматически формирует список покупок по рецептам. Пользователь может отправить текст рецепта, ссылку на кулинарный сайт или фотографию рецепта, после чего бот извлекает ингредиенты, приводит их к единому виду и объединяет одинаковые продукты.

---

# 🎯 Какую проблему решает

При приготовлении нескольких блюд пользователю приходится вручную:

* выписывать ингредиенты;
* объединять одинаковые продукты;
* суммировать их количество;
* составлять список покупок.

Бот автоматизирует этот процесс и значительно экономит время.

---

# 👥 Для кого

# 🚀 Запуск проекта

## 1. Установить зависимости

```bash
pip install aiogram openai aiohttp beautifulsoup4 python-dotenv Pillow
```

## 2. Создать файл `.env`

Скопировать `.env.example` в `.env` и заполнить:

```text
TELEGRAM_TOKEN=токен_бота
OPENROUTER_API_KEY=API_ключ
```

## 3. Запустить проект

```bash
python tg.py
```

# 📥 Входные данные

Бот поддерживает три варианта ввода:

| Входные данные       | Описание                                           |
| -------------------- | -------------------------------------------------- |
| 📝 Текст рецепта     | Пользователь отправляет рецепт сообщением          |
| 🔗 Ссылка            | Бот извлекает рецепт со страницы                   |
| 📷 Фото или скриншот | Vision-модель распознаёт ингредиенты с изображения |

---

# 📤 Результат

После обработки бот возвращает:

* название блюда;
* список ингредиентов;
* итоговый список покупок;
* объединённые одинаковые продукты с суммарным количеством.

# 💡 Пример работы

### Ввод

```text
Как приготовить пасту "Карбонара":
Нарезать грудинку мелкими кубиками.
Разогреть сковороду, растопить 1 ст. ложку сливочного масла и отправить туда грудинку. Добавить 2 зубчика чеснока, предварительно раздавленных ножом. Грудинку обжаривать на маленьком огне 10 минут, из неё должен вытопиться жир, но она должна остаться мягкой. После обжаривания чеснок убрать.
В кипящую воду добавляем 1 ст. ложку соли, 1 ст. ложку оливкового масла и спагетти. Варим так, как указано на упаковке спагетти, - важно соблюдать указанное время варки, чтобы сварить спагетти правильно.
У трёх яиц отделить белки от желтков. Отправить в миску 3 желтка и одно яйцо с белком, посолить и поперчить (по 1 щепотке), хорошо взбить и добавить 2 ст. ложки натертого сыра. Перемешать.
```

### Результат

```text
📋 Найдены ингредиенты:
────────────────────
🔹 Грудинка — 1 шт
🔹 Масло сливочное — 2 ст. л.
🔹 Чеснок — 2 шт
🔹 Соль — 1 ст. л.
🔹 Соль — 3 г
🔹 Масло оливковое — 1 ст. л.
🔹 Спагетти — 1 шт
🔹 Яйцо — 4 шт
🔹 Перец чёрный молотый — 3 г
🔹 Сыр — 4 ст. л.
────────────────────
Сохранить как «Паста Карбонара»?
Или введи своё название:
```

Если затем добавить ещё один рецепт, содержащий молоко и яйца, бот автоматически объединит одинаковые ингредиенты и увеличит их количество.

# 🤖 Роль искусственного интеллекта

ИИ используется для интеллектуального анализа рецептов.

Модель выполняет следующие задачи:

* распознаёт рецепты из текста, ссылок и изображений;
* определяет название блюда;
* выделяет ингредиенты;
* приводит названия продуктов к единому виду;
* переводит ингредиенты на русский язык;
* определяет приблизительное количество при отсутствии точных данных;
* возвращает результат в формате JSON для дальнейшей обработки.

# ⚠ Ограничения

* требуется подключение к Интернету;
* необходим действующий API-ключ провайдера LLM;
* качество распознавания фотографий зависит от качества изображения;
* сложные или неоднозначные рецепты могут потребовать проверки пользователем;
* бот предназначен для обработки рецептов, содержащих список ингредиентов.
